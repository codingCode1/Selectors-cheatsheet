CSS Selectors

Selectors

Example Selector Meaning
* { } Universal Selector Applies to all elements in the document.
* { } Targets all elements on the page.
h1,h2,h3 { } Type Selector Matches element names.
Targets the <h1>, <h2> and <h3> elements.
.note{ } p.note{ } Class Selector Matches an element whose class attribute has a value that matches the one specified after the period(or full stop) symbol.
.note{ }Targets any element whose class attribute has a value of note.
p.note{ } Targets only <p> elements whose class attribute has a value of note.
#introduction{ } ID Selector Matches an element whose id attribute has a value that matches the one specified after the pound or hash symbol.
#introduction{ } Targets the element whose id attribute has a value of introduction
li>a { } Child Selector Matches an element that is a direct child of another.
li>a { } Targets any <a> elements that are children of an <li> element (but no other <a> elements in the page).
p a { } Descendant Selector Matches and element that is a descedent of another specified element (not just a direct child of that element).
p a { } Targets any <a> elements that sit inside a <p> element, even if there are other elements nested between them.
h1+p { } Adjacent Sibling Selector Matches an element that is the next sibling of another.
h1+p { } Targets the first <p> element after any <h1> element (but not other <p> elements)
h1~p { } General Sibling Selector Matches an element that is a sibling of another, although it does not have to be the directly preceding element.
h1~p { } If you had two <p> elements that are siblings of an <h1> element, this rule would apply to both.

Attribute Selectors

Example Selector Meaning
p[class] Existence [ ] Matches a specific attritube (whatever its value).
Targets any <p> element with an attribute called class.
p[class="dog"] Equality [=] Matches a specific attribue with a specific value.
Targets any <p> element with an attribure called class whose value is dog.
p[class~="dog"] Space [~=] Matches a specific attribute whose value appears in a space-separeted list of words.
Targets any <p> element with an attribute called class whose value is a list of space-separated words, one of which is dog.
p[attr^="d"] Prefix [^=] Matches a specific attribute whose value begins with a specific string.
Targets any <p> element with an attribute whose value begins with the letter "d".
p[attr*="do"] Substring [*=] Matches a specific attribute whose value contains a specific substring.
Targets any <p> element with an attribute whose value contains the letters "do".
p[attr$="g"] Suffix [$=] Matches a specific attribute whose value ends with a specific string.
Targets any <p> element with an attribute whose values ends with the letter "g".
